# 注意
每个人创建自己的文件夹xxx_project，在自己的文件夹目录下开发项目
# 开始 
在自己的文件夹目录下simplifiedTikTok/xxx_project执行：go mod init github.com/xxx/simplifiedTikTok
然后在simplifiedTikTok/目录下执行：go work use xxx_project
# 接下来回到自己的项目路径下开发即可
